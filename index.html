Таблица с данными оглавления.
Цель работы Описание лабораторной установки Порядок выполнения работы
Изучить принципы построения HTML-документа, его структуру и форматирование. Рассмотреть возможности форматирования текста, организации списков, использование гиперссылок, форм и графики в WEB-документах. Понятие табличного представления данных. Познакомиться с системой контроля версий (СКВ) Git и возможностями сервиса Github Pages. Для выполнения лабораторной работы рекомендуется использовать следующее свободное программное обеспечение: редактор кода MS Visual Studio Code или любой другой, знакомый пользователю и имеющий подсветку синтаксиса (Notepad++, Sublime Text, Atom, Brackets и т.п.); СКВ Git (для работы с сервисом GitHub Pages потребуется доступ к сети Интернет). Создать HTML-документ на произвольную тематику на основе произвольного макета страницы (одноколоночного, 2-х или 3-х колоночного, пример структуры страницы с двумя колонками). Основной контент страницы должен содержать блочные и строковые элементы, заголовки. Содержание произвольное. Можно использовать табличную верстку ‐ в целях лучшего освоения HTML и понимания недостатков этого способа построения страницы. В целом, данную лабораторную работу выполнять пока без использования CSS. Применение семантической вёрстки вместо табличной (по желанию!), в случае простой одноколоночной структуры страницы, возможно и без управления размещением блоков средствами CSS.
